# Interline Help Center

Welcome to the Interline documentation. Interline is a unified messaging platform that lets your team handle all client communication — **SMS, WhatsApp, and email** — from one place.

This help center is organized for three audiences:

- **[Agent Guide](agent/index.md)** — day-to-day Inbox work: handling conversations, sorting, canned replies, comments, opening/closing, and notifications.
- **[Admin Guide](admin/index.md)** — setup, team management, roles, channels, contacts, campaigns (Broadcast & Keywords), auto-responders, and billing.
- **[API & Reference](api/index.md)** — developer documentation (coming soon).

## The three core tools

Interline is built around three components. Start with whichever matches what you need to do.

| Tool | What it's for | Guide |
|------|---------------|-------|
| **Inbox** | Handle every incoming client message across all channels in one shared workspace. | [Agent Guide →](agent/index.md) |
| **Broadcast** | Send SMS or WhatsApp campaigns to lists of contacts. | [Broadcast →](broadcast/index.md) |
| **Keywords** | Auto-reply to clients based on the words they text in. | [Keywords →](keywords/index.md) |

## New to Interline?

Start with the [Getting Started](getting-started.md) guide for a quick tour. If you're an agent, head to the [Agent Guide](agent/index.md) — the Inbox is where most of your day will happen. If you're setting things up, see the [Admin Guide](admin/index.md).
