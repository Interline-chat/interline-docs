# Interline Help Center

Welcome to the Interline documentation. Interline is a unified messaging platform that lets your team handle all client communication — **email, SMS, and WhatsApp** — from one place.

This help center is organized for three audiences:

- **Agents & users** — day-to-day guides for handling conversations, sending campaigns, and setting up automated replies.
- **Admins** — setup, team management, roles, billing, and channel configuration.
- **Developers** — API and reference documentation (coming soon).

## The three core tools

Interline is built around three components. Start with whichever matches what you need to do.

| Tool | What it's for | Guide |
|------|---------------|-------|
| **Inbox** | Handle every incoming client message across all channels in one shared workspace. | [Inbox →](inbox/index.md) |
| **Broadcast** | Send SMS or WhatsApp campaigns to lists of contacts. | [Broadcast →](broadcast/index.md) |
| **Keywords** | Auto-reply to clients based on the words they text in. | [Keywords →](keywords/index.md) |

## New to Interline?

Start with the [Getting Started](getting-started.md) guide for a quick tour, then dive into the [Inbox](inbox/index.md) — it's where most of your day will happen.
