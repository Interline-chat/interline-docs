# Admin Guide

The Admin Guide covers everything used to **set up and run** an Interline workspace — channels, team and roles, contacts, automation, campaigns, and message settings. These are tasks for **admins and managers**, not the day-to-day work agents do in the Inbox.

If you're a front-line agent handling conversations, see the [Agent Guide](../agent/index.md) instead.

## What's here

| Area | What it covers |
|------|----------------|
| [Company Settings](company-settings.md) | Organization info, timezone, address, and business hours. |
| [Channels](channels.md) | Connecting and managing SMS, WhatsApp, and Email channels. |
| [Users & Roles](users-roles.md) | Adding users, assigning roles (Admin / Manager / Agent) and inbox access. |
| [Inboxes](inboxes.md) | Creating shared inboxes (name, color, members). |
| [Custom Views](custom-views.md) | Saved, filtered conversation views (private or shared). |
| [Importing Contacts](contacts-import.md) | Bulk CSV import (with field mapping) and adding single contacts. |
| [Audiences](audiences.md) | Manual and dynamic lists you send campaigns to. |
| [Custom Fields & Tags](custom-fields.md) | Custom contact fields and the shared tag list. |
| [Canned Replies](canned-messages.md) | Building the shared library of pre-written replies (with {variables}). |
| [Broadcast (Campaigns)](../broadcast/index.md) | Sending SMS / WhatsApp campaigns to lists. |
| [Automation](automation.md) | Auto-assign routing, auto-responders, and keyword campaigns. |
